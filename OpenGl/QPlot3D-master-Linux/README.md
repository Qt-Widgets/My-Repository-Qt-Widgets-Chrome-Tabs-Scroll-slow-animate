# QPlot3D

QPlot3D is intended to be a lightweight 3D plot library for QT based on OpenGL.

## Example

```c++
  QCurve3D curve();
  curve.addData(0,0,0);
  curve.addData(QVector3D(1,1,1);
  curve.setColor(Qt::green);

  QPlot3D plot;
  plot.addCurve(&curve);
  
  plot.show();
```
